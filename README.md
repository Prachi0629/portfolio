

# 🚀 Prachi Joshi - Developer Portfolio

A modern, responsive, and interactive personal portfolio website built using **React.js**, **Vite**, and **Tailwind CSS** to showcase my skills, projects, education, experience, and achievements.

🔗 Live Portfolio: https://portfolio-prachijoshi.vercel.app/

---

## 📌 About

This portfolio serves as a digital representation of my technical journey, highlighting my expertise in:

- Full Stack Web Development
- DevOps & Cloud Technologies
- Data Structures & Algorithms
- Database Management
- Modern Frontend Development
- Backend API Development

The website is fully responsive and optimized for desktop, tablet, and mobile devices.
